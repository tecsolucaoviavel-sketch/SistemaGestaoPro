# 🚀 SistemaGestaoPro (Pro Management System)

A real-world Inventory and Stock Management System developed entirely on a mobile device using **Termux**, **Python**, and **SQL**.

## 📊 Core Features
- **Inventory Management:** Full registration of products, prices, and stock levels.
- **Relational Database:** Real SQL structure for data persistence.
- **Interactive CLI:** A user-friendly command-line interface for business operations.

## 🛠️ Tech Stack
- **Language:** Python 3
- **Database:** SQLite3
- **Environment:** Android (Termux)
- **Version Control:** Git & GitHub

## 🎯 Project Goal
This project demonstrates the power of mobile development to solve real business problems, focusing on clean code and database integrity.
